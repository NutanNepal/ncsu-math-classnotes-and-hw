\question{
    Let $A$ and $B$ be rings and $f:A\to B$ a surjective homomorphism.
    \begin{enumerate}
        \item[a.]
              Prove that $f(\rad A)\subset \rad B$, and construct
              an example where the inclusion is strict.\\
              \hspace*{10mm} - If $x\in\rad A$, then $f(1+Ax)=1+Bf(x)$ should
              be a set of units of B.
        \item[b.]
              Prove that if $A$ is a semilocal ring then $f(\rad A)= \rad B$.
    \end{enumerate}
}
