\question{
    Define $l^p$ as the set of all sequences $x=\{x_i\}_{1}
    ^{\infty}$ such that $\left(\sum_{1}^{\infty}
    |x_i|^{p}\right)^{\frac{1}{p}}<\infty$ and define the
    metric $d$ in $l^p$ by
    $$d_{p}(x,y)=\left(\sum_{1}^{\infty}{|x_i-y_i|^{p}}\right)
    ^{\frac{1}{p}}\hspace{10mm}x,y\in l^p.$$
    Prove that $l^p$ is a metric space.
}
